Instalación OpenMediaVault

  • luismc
  • 22 de Octubre del 2021

Openmediavault es un software para sistemas basados en Debian, con el que podremos montar nuestro propio NAS con la configuración hardware de nuestra casi total elección.

Los requisitos mínimos actualmente son:

  • Procesador x86 de 64 bits o procesadores ARM compatibles.
  • 1 GB de memoria RAM.
  • Un disco para el sistema de al menos 4 GB.
  • Uno o más discos para las necesidades de almacenamiento.

Hay dos métodos de instalación. La primera mediante imagen ISO pre-configurada y la segunda mediante script.

La primera no tiene más misterio que la típica instalación desde cd/usb.

La segunda es desde un sistema previamente instalado ejecutar un script facilitado por los desarrolladores para instalar el software sobre el.

La principal diferencia y ventaja de este ultimo método es que te servirá para instalar openmediavault y los omv-extras.

Esto mejora muchísimo las capacidades del sistema con plugins adicionales y la posibilidad de instalar Docker en el a golpe de clic entre otros.

También mencionar que este script es el método recomendado para equipos Raspberry.

Todo lo anterior con una sola linea de terminal, lo que demuestra el gran trabajo del equipo de desarrolladores.

INSTALACIÓN OPENMEDIAVAULT MEDIANTE SCRIPT

sudo wget -O - https://github.com/OpenMediaVault-Plugin-Developers/installScript/raw/master/install | sudo bash
