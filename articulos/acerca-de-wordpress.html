¿De que va?

WordPress es el gestor de contenido más extendido del mundo y no por nada.

Sus principales virtudes son su facilidad de uso y enorme comunidad detrás. Esto se puede traducir en miles de plugins y temas para casi cualquier objetivo.

Es utilizado por empresas grandes, pequeño comercio, particulares y básicamente cualquier tipo de entidad.

Se puede utilizar para la creación ya sea de páginas web, tiendas online e incluso webs corporativas.

Su facilidad de uso hace que sea la plataforma preferida para personas sin conocimientos en desarrollo web así como proyectos rápidos y ‘pequeños’. Aunque cabe decir que es un herramienta tan versátil que vale para cualquier tipo de finalidad que puedas tener siempre y cuando conozcas un poco las tecnologías que hay detrás.

Estás tecnologías son:

  • La Santa Trinidad del mundillo web html, css y JavaScript.
  • php como lenguaje de servidor.
  • MySql como base de datos.
  • apache típicamente como servidor web.

Luego está la API que desarrollaron y que expande las posibilidades a casi cualquier integración que se te pueda ocurrir a niveles muy altos.

Alternativas

Cómo alternativas surgieron Joomla, Ghost, Textpattern, Statamic, Subrion, Shopify, drupal y un largo etcétera.

Todas tienes sus virtudes, particularidades y comunidad detrás. Por lo que como con cualquier herramienta, el que sea mejor o peor solo lo determina la pericia de quien la utiliza y el entorno en el que se moverá.

Cualquier herramienta tiene su función y no es lo mismo pintar un coche con pistola que con un spray, ambas pintan pero no igual dependiendo de la superficie ya que cada una tiene su objetivo para el que fue diseñado.

Mi opinión

Para mi lo que más me gusta de este software es su api y la parte de gestor que con sus miles de plugins te ayudan a poder crear contenido, redactarlo, organizarlo y difundirlo rápidamente en redes sociales si te creas un buen flujo de trabajo.

De hecho lo utilizo en esta web para eso mismo, ayudarme con la redacción, organización y posterior difusión (o por lo menos lo intenta 😉 aunque ésta sea una web estática.
